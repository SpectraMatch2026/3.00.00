\chapter{SpectraMatch Erişim ve İndirme Bağlantıları}
\label{app:spectramatch-erisim}

\noindent Bu ek, \textit{SpectraMatch} yazılımının resmî web uygulamasına, teknik veri sayfalarına ve örnek analiz çıktılarına (PDF) doğrudan erişim amacıyla düzenlenmiştir. Aşağıdaki bağlantılar, tez kapsamında tekrarlanabilirlik ve şeffaflık ilkeleri doğrultusunda tek bir yerde toplanmıştır.

\paragraph*{Genel bağlantılar (özet erişim).}
\begin{itemize}
  \item Resmî web uygulaması: \url{https://spectramatch.com.ly}
  \item Ürün veri sayfaları (İngilizce/Türkçe PDF girişi): \url{https://spectramatch.com.ly/datasheets}
  \item Türkçe teknik veri sayfası (PDF): \url{https://spectramatch.com.ly/datasheetTR.pdf}
  \item İngilizce teknik veri sayfası (PDF): \url{https://spectramatch.com.ly/datasheetEN.pdf}
  \item Örnek analiz raporları merkezi (çok dilli arayüz): \url{https://spectramatch.com.ly/reports/}
\end{itemize}

\paragraph*{Örnek PDF çıktıları (dosya düzeyinde doğrudan indirme).}
Aşağıdaki dosya adları, barındırma alanında standart yazım ile yayınlanmıştır (yayın ortamı büyük/küçük harfe duyarlı olabilir).
\begin{itemize}
  \item Birleşik (tam) analiz raporu: \url{https://spectramatch.com.ly/Full_Report.pdf}
  \item Renk analiz raporu: \url{https://spectramatch.com.ly/Color_Report.pdf}
  \item Desen analiz raporu: \url{https://spectramatch.com.ly/Pattern_Report.pdf}
  \item Ayarlar özeti (oturum makbuzu): \url{https://spectramatch.com.ly/Settings_Receipt.pdf}
\end{itemize}

\paragraph*{Tarayıcı içi görüntüleme sayfaları (PDF + gezinme çubuğu).}
Aynı belgeler, kullanıcı arayüzü ile birlikte yeni bir oturumda incelenmek üzere aşağıdaki yollardan da açılabilir:
\begin{itemize}
  \item \url{https://spectramatch.com.ly/fullreport/}
  \item \url{https://spectramatch.com.ly/colorreport/}
  \item \url{https://spectramatch.com.ly/patternreport/}
  \item \url{https://spectramatch.com.ly/settingsreport/}
\end{itemize}

\noindent\textit{Not:} Windows masaüstü sürümü (\textbf{v3.0.0}) resmî web sitesi üzerinden indirilebilir. Teknik veri sayfaları üzerinden sistemin temel modülleri, kullanım kapsamı ve çıktı yapısı; örnek rapor bağlantıları üzerinden ise üretilen PDF rapor türleri hakkında ek doğrulama sağlanabilir.
